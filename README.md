# Consulta de CEP com Validação de Access Token JWT - .NET 7

Este projeto em .NET 7 permite realizar consultas de CEP utilizando um serviço que valida Access Tokens JWT com Azure AD B2C.

## Descrição

O objetivo principal deste projeto é demonstrar como integrar a validação de Access Tokens JWT emitidos pelo Azure AD B2C em um serviço de consulta de CEP. Isso permite controlar o acesso ao serviço utilizando autenticação baseada em tokens, proporcionando maior segurança.

## Pré-requisitos

* **.NET 7 SDK:** Certifique-se de ter o SDK do .NET 7 instalado na sua máquina.
* **Conta Azure AD B2C:** É necessário ter uma conta no Azure AD B2C e configurar um aplicativo para emissão de tokens.

## Uso

Para utilizar o serviço de consulta de CEP protegido por autenticação via Azure AD B2C, siga os passos:

1. **Autenticação:**
   Faça a autenticação e obtenha um Access Token JWT do Azure AD B2C.
2. **Chamada à API:**
   Utilize o Access Token obtido para fazer chamadas à API de consulta de CEP, enviando-o no cabeçalho de autorização (`Authorization: Bearer <access-token>`).
